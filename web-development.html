A responsive weather application that retrieves real-time weather data using the OpenWeather API. The app allows users to search for any city and instantly displays current conditions like temperature and location details.

Key Features

  • Real-time Data: Direct integration with OpenWeather API.
  • Dynamic UI: Interface updates instantly based on search results.
  • Responsive Design: Fully optimized for mobile, tablet, and desktop.
  • Clean UX: Minimalist interface focused on speed and usability.
